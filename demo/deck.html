Autonomous

It does not wait to be asked

Background watcher
Scans the ledger on a loop. A new spike triggers an investigation with no human in the loop.
It takes action
Opens a GitHub issue, tightens a rate limit, or writes a corrected file and opens a fix pull request.
Real GitHub actions
The agent reads PR diffs and source files, then ships a real branch and PR to your repo.
Every action logged
The dashboard shows each scan, attribution, and remediation as it happens, live.
